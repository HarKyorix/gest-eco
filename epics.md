# 📊 Epics & User Stories - Family-Eco

## Epic 1: Gestion des Boards et Plannings 🗂️

### Description
Permettre aux utilisateurs de créer et organiser leurs budgets par boards (périodes/projets) avec des plannings multiples.

### User Stories

- [ ] En tant qu'utilisateur, je veux créer un nouveau board pour organiser mes budgets par période (mensuel, annuel, projet)
- [ ] En tant qu'utilisateur, je veux modifier le titre et la description d'un board existant
- [ ] En tant qu'utilisateur, je veux supprimer un board avec confirmation pour éviter les erreurs
- [ ] En tant qu'utilisateur, je veux créer plusieurs plannings au sein d'un board pour gérer différents scénarios budgétaires
- [ ] En tant qu'utilisateur, je veux éditer le titre d'un planning en ligne pour une modification rapide
- [ ] En tant qu'utilisateur, je veux réorganiser mes plannings par glisser-déposer (drag & drop)
- [ ] En tant qu'utilisateur, je veux dupliquer un planning pour créer rapidement un scénario similaire
- [ ] En tant qu'utilisateur, je veux archiver un board sans le supprimer pour garder l'historique

## Epic 2: Gestion des Budgets 💰

### Description
Permettre aux utilisateurs d'allouer et suivre leurs budgets par source de revenu.

### User Stories

- [ ] En tant qu'utilisateur, je veux ajouter un budget avec un montant et une source de revenu
- [ ] En tant qu'utilisateur, je veux modifier le montant d'un budget en ligne
- [ ] En tant qu'utilisateur, je veux ajouter un commentaire à un budget pour contextualiser l'allocation
- [ ] En tant qu'utilisateur, je veux supprimer un budget avec notification de confirmation
- [ ] En tant qu'utilisateur, je veux voir le total des budgets alloués dans un planning
- [ ] En tant qu'utilisateur, je veux rechercher un budget par nom de source
- [ ] En tant qu'utilisateur, je veux trier mes budgets par montant (croissant/décroissant)
- [ ] En tant qu'utilisateur, je veux trier mes budgets par nom de source (A-Z / Z-A)
- [ ] En tant qu'utilisateur, je veux voir une alerte visuelle colorée indiquant l'état de mon budget (dépassé/équilibré/faible)
- [ ] En tant qu'utilisateur, je veux voir le pourcentage et le montant restant de mon budget
- [ ] En tant qu'utilisateur, je veux être averti quand mon budget est dépassé (alerte rouge)
- [ ] En tant qu'utilisateur, je veux être averti quand mon budget est faible < 20% (alerte orange)
- [ ] En tant qu'utilisateur, je veux sélectionner plusieurs budgets pour des actions groupées (suppression, export)

## Epic 5: Gestion des Catégories (Sources, Caisses, Divers) 🏷️

### Description
Permettre aux utilisateurs de personnaliser leurs catégories de sources de revenus, caisses d'épargne et types de dépenses.

### User Stories

#### Sources de Revenus
- [ ] En tant qu'utilisateur, je veux créer une nouvelle source de revenu (salaire, freelance, investissement)
- [ ] En tant qu'utilisateur, je veux modifier le nom d'une source de revenu
- [ ] En tant qu'utilisateur, je veux supprimer une source de revenu non utilisée
- [ ] En tant qu'utilisateur, je veux voir quels budgets utilisent une source avant de la supprimer

#### Caisses d'Épargne
- [ ] En tant qu'utilisateur, je veux créer une nouvelle caisse d'épargne (vacances, urgence, projet)
- [ ] En tant qu'utilisateur, je veux modifier le nom d'une caisse
- [ ] En tant qu'utilisateur, je veux supprimer une caisse non utilisée
- [ ] En tant qu'utilisateur, je veux voir le solde total d'une caisse (épargnes - dépenses)

#### Catégories de Dépenses (Divers)
- [ ] En tant qu'utilisateur, je veux créer une nouvelle catégorie de dépense (alimentation, transport, loisirs)
- [ ] En tant qu'utilisateur, je veux modifier le nom d'une catégorie de dépense
- [ ] En tant qu'utilisateur, je veux supprimer une catégorie non utilisée
- [ ] En tant qu'utilisateur, je veux voir combien de dépenses utilisent une catégorie

## Epic 7: Recherche, Filtrage et Tri 🔍

### Description
Permettre aux utilisateurs de trouver rapidement des informations spécifiques dans leurs données financières.

### User Stories

- [ ] En tant qu'utilisateur, je veux rechercher en temps réel dans mes budgets par nom de source
- [ ] En tant qu'utilisateur, je veux rechercher en temps réel dans mes dépenses par catégorie
- [ ] En tant qu'utilisateur, je veux rechercher en temps réel dans mes épargnes par caisse

## Epic 8: Historique et Annulation (Undo/Redo) ↩️

### Description
Permettre aux utilisateurs d'annuler et de rétablir leurs actions pour éviter les erreurs.

### User Stories

- [ ] En tant qu'utilisateur, je veux annuler ma dernière action (Ctrl+Z / Cmd+Z)
- [ ] En tant qu'utilisateur, je veux rétablir une action annulée (Ctrl+Y / Cmd+Shift+Z)
- [ ] En tant qu'utilisateur, je veux voir si l'undo/redo est disponible visuellement (boutons actifs/inactifs)
- [ ] En tant qu'utilisateur, je veux que l'historique conserve 50 actions maximum
- [ ] En tant qu'utilisateur, je veux recevoir une notification quand j'utilise undo/redo
- [ ] En tant qu'utilisateur, je veux que les raccourcis clavier s'ignorent quand je tape dans un champ texte
- [ ] En tant qu'utilisateur, je veux fermer les formulaires avec la touche Escape
- [ ] En tant qu'utilisateur, je veux que l'historique capture les ajouts, modifications et suppressions de budgets
- [ ] En tant qu'utilisateur, je veux que l'historique capture les ajouts, modifications et suppressions de dépenses
- [ ] En tant qu'utilisateur, je veux que l'historique capture les ajouts, modifications et suppressions d'épargnes

## Epic 9: Export et Import de Données 💾

### Description
Permettre aux utilisateurs de sauvegarder et restaurer leurs données pour éviter la perte d'informations.

### User Stories

- [ ] En tant qu'utilisateur, je veux exporter toutes mes données en format JSON
- [ ] En tant qu'utilisateur, je veux que le fichier exporté porte un nom avec la date (backup-YYYY-MM-DD.json)
- [ ] En tant qu'utilisateur, je veux importer un fichier de sauvegarde JSON
- [ ] En tant qu'utilisateur, je veux que l'import valide la structure du fichier avant de charger les données
- [ ] En tant qu'utilisateur, je veux recevoir une notification de succès après un export
- [ ] En tant qu'utilisateur, je veux recevoir une notification de succès après un import valide
- [ ] En tant qu'utilisateur, je veux recevoir une alerte d'erreur si l'import échoue avec le détail de l'erreur
- [ ] En tant qu'utilisateur, je veux que l'export inclue tous mes boards, plannings, caisses, sources et catégories
- [ ] En tant qu'utilisateur, je veux que l'export inclue mes préférences utilisateur
- [ ] En tant qu'utilisateur, je veux pouvoir importer mes données sur un autre navigateur ou appareil

## Epic 10: Notifications et Feedback Utilisateur 🔔

### Description
Fournir un feedback clair et non-intrusif à l'utilisateur pour toutes ses actions.

### User Stories

- [ ] En tant qu'utilisateur, je veux recevoir une notification toast de succès (✅) quand j'ajoute un élément
- [ ] En tant qu'utilisateur, je veux recevoir une notification toast de succès (✅) quand je modifie un élément
- [ ] En tant qu'utilisateur, je veux recevoir une notification toast de succès (✅) quand je supprime un élément
- [ ] En tant qu'utilisateur, je veux recevoir une notification toast d'erreur (❌) quand une action échoue
- [ ] En tant qu'utilisateur, je veux recevoir une notification toast d'information (ℹ️) pour les actions neutres
- [ ] En tant qu'utilisateur, je veux recevoir une notification toast d'avertissement (⚠️) pour les alertes
- [ ] En tant qu'utilisateur, je veux que les notifications disparaissent automatiquement après 4 secondes
- [ ] En tant qu'utilisateur, je veux pouvoir fermer manuellement une notification en cliquant dessus
- [ ] En tant qu'utilisateur, je veux voir plusieurs notifications empilées si plusieurs actions se produisent
- [ ] En tant qu'utilisateur, je veux que les notifications soient visuellement adaptées au mode sombre/clair

### Description
Garantir l'intégrité des données en validant les entrées utilisateur.

### User Stories

- [ ] En tant qu'utilisateur, je veux qu'on m'empêche de saisir un montant nul ou négatif
- [ ] En tant qu'utilisateur, je veux voir un message d'erreur clair "Montant doit être supérieur à 0"
- [ ] En tant qu'utilisateur, je veux qu'on m'empêche de créer un budget sans source de revenu
- [ ] En tant qu'utilisateur, je veux qu'on m'empêche de créer une épargne sans caisse
- [ ] En tant qu'utilisateur, je veux qu'on m'empêche de créer une dépense sans catégorie si elle ne puise pas dans une caisse
- [ ] En tant qu'utilisateur, je veux qu'on m'empêche de supprimer une source utilisée dans des budgets
- [ ] En tant qu'utilisateur, je veux qu'on m'empêche de supprimer une caisse utilisée dans des épargnes/dépenses
- [ ] En tant qu'utilisateur, je veux qu'on m'empêche de supprimer une catégorie utilisée dans des dépenses
- [ ] En tant qu'utilisateur, je veux recevoir une confirmation avant de supprimer un board/planning
- [ ] En tant qu'utilisateur, je veux que les champs de montant acceptent les décimales (0.01)

### User Stories

### Description

### User Stories

- [ ] En tant qu'utilisateur, je veux que toutes mes données restent locales dans mon navigateur
- [ ] En tant qu'utilisateur, je veux qu'aucune donnée ne soit envoyée à un serveur distant
- [ ] En tant qu'utilisateur, je veux pouvoir effacer toutes mes données localement

- [ ] En tant qu'utilisateur, je veux que mes données exportées soient en format lisible (JSON)
- [ ] En tant qu'utilisateur, je veux contrôler quand et où mes données sont exportées
